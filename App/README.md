# YOLOv8 Prediction GUI

A desktop application for image-based object detection or segmentation using a pretrained YOLOv8 model. This tool is intended to provide a user-friendly interface to run predictions on batches of images and visually compare results.

---

## 🚀 Features

- Upload and process multiple images at once
- Automatically clear previous results before each prediction
- Side-by-side comparison between original and predicted images
- Simple and lightweight PyQt5-based interface
- Supports both `detect` and `segment` models

---

## 🛠 Requirements

- **Python 3.9.21**  
- Install required libraries via `requirements.txt`:

```bash
pip install -r requirements.txt
```

---

## How to run
```bash
python app.py
```
